# Canvapi

API documentation with canvas visualization — drop-in replacement for Swagger UI and Scalar.

## What is Canvapi?

Canvapi renders your OpenAPI spec as an interactive canvas dashboard instead of a vertical list. Built for both humans and AI.

- **Canvas visualization** — endpoints, schemas, and relationships as interactive nodes
- **Dashboard layout** — customizable blocks inspired by Business Model Canvas
- **AI-ready** — `llms.txt` endpoint for LLMs to understand your API
- **Microservices** — unified view of multiple APIs and their connections
- **Drop-in replacement** — works with FastAPI, Express, NestJS, and more
